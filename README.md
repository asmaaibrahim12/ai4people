# AI4People

Companion website for **Making AI Work for People** by Asmaa Ibrahim, published by Wiley.

## Pages

- **The Book** (`artifacts.html`) — overview, PRESS Framework, downloadable artifacts, notify signup
- **About** (`author.html`) — author bio, speaking, advisory, and workshop info
- **Contact** (`contact.html`) — contact form powered by Web3Forms

## Tech

Static HTML/CSS site hosted on GitHub Pages at [ai4people.tech](https://ai4people.tech). No build step required.

## Run locally

Open `index.html` in a browser — it redirects to the Book page.
